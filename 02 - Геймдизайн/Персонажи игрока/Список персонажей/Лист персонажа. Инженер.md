**Тактика персонажа:**
-Персонаж имеет мало брони. Из-за этого персонаж постоянно теряет здоровье. Но персонаж имеет вампиризм по умолчанию. Данная система заставляет игрока действовать более активно, атакуя врагов. 


**Характеристика персонажа:**

| Характеристика персонажа      | Значение |
| ----------------------------- | -------- |
| ХП (Количество здоровья)      | 15       |
| Броня                         | 3        |
| Резист                        | 13%      |
| Энергия                       | 5        |
| Шаг                           | 4        |
| Количество слотов под амулеты | 6        |
