Обычные объекты, что занимают одну клетку:

| **Название клетки**     | **Описание клетки**                                                                                                                                                                                                                           |
| ----------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Клетка высадки          | Обозначает, куда в следующем ходу приземлятся вражеские юниты. Если игрок наступает на клетку, то враг не спавнится, а персонаж игрока получает урон по ХП.                                                                                   |
| Баллон азота            | Накладывает эффект на юнитов в этих клетках «Дым». Юниты под этим эффектом не могут стрелять и кидать снаряды. Тушит огненные клетки.                                                                                                         |
| Дым                     | Накладывает эффект на юнитов в этих клетках «Дым». Юниты под этим эффектом не могут стрелять и кидать снаряды.                                                                                                                                |
| НПС                     | Игроку нужно защитить НПС. Он не должен для этого получать урон в течении всего прохождения. За спасение дает деньги.                                                                                                                         |
| Ленточная клетка        | Перемещает юнита в направлении, куда указывает клетка. Может быть уничтожена.                                                                                                                                                                 |
| ЭМИ-мина                | Поражает ЭМИ-атакой юнита, когда он проходит по данной клетке.                                                                                                                                                                                |
| Противотанковая мина    | Уничтожает юнита с любым количеством здоровья, когда он проходит эту клетку.                                                                                                                                                                  |
| Бочка с кислотой        | При уничтожении бочки распрыскивает в области кислоту, накладывая на всех в радиусе попадания на юнитов статус «кислота».                                                                                                                     |
| Бочка воды              | При разрушении превращает соседние обычные клетки в водные.                                                                                                                                                                                   |
| Легкая стена            | Преграждает путь игрока. Может быть разрушена.                                                                                                                                                                                                |
| Укрепленная стена.      | Преграждает путь игрока. Не может быть разрушена. Имеет скрытый шанс, который выставляется стене в начале хода, с которым она может превратится в легкую стену при получении урона.                                                           |
| Окно                    | Место спавна вражеских юнитов. Если толкнуть юнита в окно – то он будет мгновенно убит.                                                                                                                                                       |
| Окно с песком           | Если разбить окно – то из него высыпется песок и клетка перед окном будет недоступна для хода. Если на клетке стоит юнит – он станится на 1 ход.                                                                                              |
| Бочка с бензином        | При выстреле в нее создает взрыв, что наносит урон и толкает. Толчок бочки или удар по ней обливает соседние клетки бензином.                                                                                                                 |
| Турель                  | Стреляет в вражеских юнитов. Можно перепрограммировать на сторону игрока.                                                                                                                                                                     |
| Сигнализация с роботами | Если вражеский юнит достигает клетки с кнопкой тревоги – то на карте просыпаются 1-2 вражеских робота. Игрок может взломать эту кнопку и поднять робота на свою сторону.                                                                      |
| Портал                  | Через порталы появляются враги на локациях. Порталы появляются на случайной стене. Условие для появления – клетка перед порталом должна быть свободна. Если клетка не свободно – то враг не появится, а объект перед порталом получит 1 урон. |
| Лазер                   | Отражает луч от поверхностей                                                                                                                                                                                                                  |
| Стена с шипами          | Толчок в стену прибивает цель к стене, цель получает стан на один ход. Наносится урон.                                                                                                                                                        |
| Гнездо                  | При разрушении гнезда из того вылезает пчела, что может отравить персонажей игрока или врагов вокруг.                                                                                                                                         |
| Стеклянная стена        | Ломается от любого удара. Персонаж может пройти сквозь него, но он получит урон. Взрыв создает ударную волну, что выбивает стекло и разбрасывает осколки, что наносят урон вокруг.                                                            |
Объекты, что могут занимать более 1 клетки:

| Название | Эффект объекта                                                                                                                                                          | Размер модели |
| -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------- |
| Лифт     | Поднимает и опускает объекты. Мгновенно уничтожает объекты, что находятся под ним.                                                                                      | 4х7           |
| Столб    | Столб на локации. При получении урона столб падает в сторону удара. Ломается, если на пути падения присутствуют объекты. Оглушает персонажей, если падает на персонажа. | 1х6           |
