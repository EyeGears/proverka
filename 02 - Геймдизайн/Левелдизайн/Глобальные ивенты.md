Случайные глобальные ивенты на локации, что оказывают влияние на геймплей:

| **Название клетки**     | **Описание клетки**                                                                                                                                               |
| ----------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Воздушный удар          | На следующий ход по данной клетке будет удар сверху                                                                                                               |
| Падающие камни          | На следующий ход по клеткам будут падать камни. Выбирается случайно.                                                                                              |
| Прилив                  | Клетки в начале хода становятся водой.                                                                                                                            |
| Удар молнии             | Случайно бьет по карте электричеством.                                                                                                                            |
| Сейсмическая активность | Сдвигает клетки в случайную сторону                                                                                                                               |
| Ледяной ветер           | Проходит на открытых локациях и через разбитые окна. Тушит объекты с эффектом «Огонь». Если долго стоит юнит на ледяном ветре, то он получает статус «заморозка». |
