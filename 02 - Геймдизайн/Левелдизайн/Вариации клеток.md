
| **Название**      | **Описание клетки**                                                                                                                               |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| Обычная           | Не имеет эффекта.                                                                                                                                 |
| Дыра              | Персонаж падает в неё, если наступает на неё, после чего он исчезает.                                                                             |
| Треснувшая клетка | При повреждении объекта, что стоит на клетке, образуется клетка «Дыра»                                                                            |
| Кислота           | Накладывает на юнита эффект «кислота», когда юнит встает на нее                                                                                   |
| Бензин            | Накладывает на юнита эффект «Огонь», когда юнит получает урон на этой клетке.                                                                     |
| Вода              | Усиливает эффект «ЭМИ» на юнита, что стоит на этой клетке. Снимает эффект «Огонь». Способна заменять клетку «дерево», если сломать данную клетку. |
| Дерево            | Деревянная клетка. Загорается на 1 ход, если подвергается эффекту «огонь». После чего становится клеткой «Сломанная клетка»                       |
| Сломанная клетка  | Если рядом находится клетка «Вода», то становится ею.                                                                                             |
| Огненная клетка   | Когда юнит проходит эту клетку, он получает эффект «Огонь»                                                                                        |
| Ленточная клетка  | Перемещает юнита в направлении, куда указывает клетка. Может быть уничтожена.                                                                     |
| Песок             | При атаке на клетке поднимается пылевая завеса, что отключает атаки у юнита, что находится на ней. По цели нельзя попасть дальнобойной атакой.    |
| Лед               | Рядом с огненной клеткой лед становится клеткой «вода».                                                                                           |
| Пираньи           | Водная клетка, которая наносит урон юнитам, что стоят в ней. Если рядом клетка «Вода», то эти клетки также становятся клетки «Пираньи».           |
| Мазут             | Снимает горение, ЭМИ, воду. Уменьшает шаги персонажей.                                                                                            |
