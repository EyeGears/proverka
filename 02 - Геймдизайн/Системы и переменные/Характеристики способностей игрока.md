Характеристики способностей врагов

| Название характеристики | Описание                                                                        | Возможные вариации                                                  |
| ----------------------- | ------------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| Цель                    | Определяет, что является целью для способности                                  | Персонажи игрока<br>Персонажи враги<br>Себя<br>Объекты              |
| Область                 | Определяет, сколько клеток задевает способность.                                | 1 (одна клетка)<br>5 (клетка цели и соседние клетки)<br>9 (область) |
| Направление             | Определяет, в какие возможные стороны может атаковать.                          | Все 4 стороны<br>В радиусе                                          |
| Дальность               | Определяет дальность клеток, на которую можно применить способность.            | N-клеток                                                            |
| ФФ                      | Определяет, может-ли способность задеть своих союзников                         | Да<br>нет                                                           |
| Урон                    | Определяет количество урона, которое может нанести способность.                 | N-число                                                             |
| Защита                  | Определяет, сколько брони цели дает способность                                 | N-число                                                             |
| Восстановение           | Определяет, сколько ХП цели дает способность                                    | N-число                                                             |
| Пробитие                | Определяет, может-ли способность пробивать цель.                                | Да<br>Нет                                                           |
| Статус-эффект           | Определяет, накладывает-ли способность на цель статус-эффект и какой он делает. | Нет<br>(Список эффектов)                                            |
| Усиление                | Определяет, будет-ли дополнительный бафф способности.                           | Нет<br>(Список баффов)                                              |
| Эмблема                 | Рисунок способности                                                             | картинка                                                            |
